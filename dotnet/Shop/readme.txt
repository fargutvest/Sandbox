Вы являетесь разработчиком ASP.NET MVC приложения, которое управляет товарами в вашем магазине, а именно: создание, удаление, добавление, редактирование.
Характеристики товара:
• Название 
• Кол-во
• Штрих-код
Необходимо использовать следующий моменты при выполнении данной задачи:
• Удаление с помощью ajax
• Использование IoC, любой из известных вам
• Проверка данных на верность ввода от пользователя
• Написание юнит тестов
• В качестве источника данных использовать простейшую реализацию СУБД, без установки серверной части (MSSQL не подходит). Например: Microsoft Access
Свой код, необходимо выкладывать в одном из публичных репозиториев, к примеру bitbucket.org. По завершению выполнения задания необходимо выслать ссылку на репозиторий.
Одно из требований также включает частые коммиты в репозиторий.
